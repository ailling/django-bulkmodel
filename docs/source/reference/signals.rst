Signals API Reference
=========================


Bulk create signals
------------------------

``pre_bulk_create``

Fired before a bulk-create is issued

``post_bulk_create``

Fired after a bulk-create is issued



Update signals
------------------------

``pre_update``


``post_update``


``pre_update_fields``


``post_update_fields``


Copy to / from signals
------------------------

``pre_copy_from_instances``


``post_copy_from_instances``
